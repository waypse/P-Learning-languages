Animation

Qu'est ce que le JavaScript ?

Ajoutons un peu d'animation dans notre site. Le javasript est un langage qui change des langages web et qui se rapproche plus d'un véritbla langage de programmation avec des variables et autres fonctions. Il va donc permettre d'ajouter des animations, des éléments intéractifs ou bien de permettre a certains éléments de faire des choses particulières avec certaines conditions.

Et que la lumière fut

Dans l'exercice d'aujourd'hui, nous allons faire quelque chose de simple encore une fois, Nous allons faire en sorte d'allumer et d'éteindre une ampoule.

Plutôt cool non ? Pour faire cet exercice vous allez avoir besoin des deux images d'ampoule que vous pouvez retrouver Ici
Assuez vous que les images soient dans le même dossier que le fichier html que nous allons créer pour afficher notre ampoule.

Nous allons donc créer un nouveau fichier html avec la même construction que notre première page, donc avec un head un body etc etc. 

                        
<!DOCTYPE html>
<html>
<head>
    <title> Ampoule </title>
    <meta charset="utf-8">
</head>
<body>

</body>
</html>

Un petit rappel de a quoi doit ressembler votre code html

Création de notre animation

Passons a la création de notre animation, nous allons commener avec les boutons.

                        
<button onclick="document.getElementById('ampoule').src='assets/Images/LightBulbOn.png'">
Allumer ampoule
</button>

Alors c'est un peu long mais pas de panique, nous allons décortiuer cette balise petit a petit.

Button permet de définir que l'élément a l'apparence d'un bouton

onlick est une condition, elle détecte si l'élément a été cliqué. Si oui alors elle exécute ce qui est indiqué devant elle (onclick="fonction que l'on veut exécuter").

document.getElementById('myImage') va permettre de modifier les caractéristique de l'élément qui comporte l'id 'ampoule'. Dans notre cas, c'est notre image d'ampoule qui possède cet id.

src='assets/Images/LightBulbOn.png' permet de dire a otre code JS quel ce qu'il doit afficher, donc notre document.getElementById va venir récupérer l'image LightBubOn et l'afficher dans l'élément qui porte l'id myImage

Nous avons donc un premier bouton pour allumer notre ampoule, pour créer celui qui va éteindre l'ampoule, il nous suffira juste de copier coller ce code et de changer l'image que nous souhaitons afficher (dans le fichier zip que je vous ai fait télécharger vous avez deux image, LightBulbOn et LightBulbOff, c'est LightBulbOff que vous allez devoir utiliser pour éteindre l'ampoule)

Nous allons maintenant créer une image avec la balise HTML <img>

                        
<img src="Image.jpg" id="ampoule">

Voici la balise img. L'argument src permet de dire a notre balise quel image il dois afficher, un peu comme avec link pour savoir quel fichier CSS il doit utiliser. Dans notre exercice nous allons lui demander d'afficher l'ampoule éteinte donc dans src nous allons écrire src="LightBulbOff.png". Il faut aussi ajouter un id pour que notre code javasript reconaisse que c'est l'élément a modifier. C'est l'argument id qui va s'en occuper.

Et puis c'est tout ! Il vous suffit juste d'assembler ces balises ensemble, dans l'exemple en début de page j'ai mis un bouton, puis mon image et mon deuxième bouton 

                        
<button onclick="document.getElementById('ampoule').src='assets/Images/LightBulbOn.png'">
Allumer ampoule
</button>
<img src="Image.jpg" id="ampoule">
<button onclick="document.getElementById('ampoule').src='assets/Images/LightBulbOn.png'">
Eteindre ampoule
</button>

Voila a quoi ressemble le code (le chemin vers mes images est différent du votre, ne copiez collez pas ce chemin ou sinon votre page web ne trouvera pas les images.)

Il ne vous reste plus qu'a enregistrer votre code au format HTML, d'ouvrir la page web et de faire mumuse pendant des heures.
