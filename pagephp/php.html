Php est un langage très importent dans un site Internet. C'est un langage dis "Back-End", c'est a dire qu'il agit en arrière fond de notre site sans avoir des éléments graphiques. Jusqu'a présent nous avons fait du "Front-end" avec le HTML, CSS et JS c'est a dire ce que l'on montre a l'utilisateur et avec quoi il peut intéragir. Nous allons donc apprendre a transférer des informations d'une page a l'autre.

Création d'un formulaire

Pour faire passer des informations dans un site, le moyen le plus simple est le formulaire. Nous allons en créer un très simple ou nous allons rentrer notre nom, prénom et adresse mail.
Nous allons donc retourner dans notre page web et rajouter dans notre body une nouvelle balise : la balise <form>. 

                            
<form method="post" action="Nom_Du_Fichier.php">

</form>

C'est dans cette balise que nous allons rentrer tout les éléments de notre formulaire Elle possède deux arguments importents, method qui indique la façon dont le formulaire doit envoyer les infos (dans notre exemple post) et action qui indique vers quel fichier les informations doivent être envoyés, donc toutes les informations contenues dans la balise form vont être envoyés via la methode post vers le fichier php que nous avons indiqué dans l'argument action.

Donc il va falloir créer un fichier .php et le connecter a notre page web comme pour le CSS ?

Oui ! Mais nous verrons ça plus tard, nous allons pour le moment nous concentrer sur la création de notre formulaire.
Nous allons commencer en parlant de la balise principale dans un formulaire ,la balise <input> 

                        
<input type="Type_De_Champ" name="Nom_Du_Champ" />

Cette balise permet de créer un champ ou rentrer une information qui va pouvoir être ensuite envoyer a notre page PHP pour pouvoir l'afficher ou la stocker. C'est une balise très polyvalente grâce a l'argument type qui permet de changer le contenu de la balise pour pouvoir contenir un certain type d'info
Voici quelques exemple de type d'input :
-text pour écrire un texte simple
-password pour écrire un mot de passe qui sera caché quand il sera écrit dans le champ
-email pour écrire une adresse mail, le navigateur saura que l'utilisateur doit écrire une adresse valide
-tel pour écrire un numéro de téléphone

Et encore plein d'autres, mais pour notre formulaire nous allons utiliser deux type d'input : le text et le email.

Trève de blabla, passons a la création de notre formulaire ! Nous voulons donc un formulaire qui va récupérer le nom prénom et email de notre utilisateur. 

                        

<form method="post" action="formulaire.php">
    <input type="text" name="Nom" />
    <br>
    <input type="text" name="prenom" />
    <br>
    <input type="email" name="mail" />
</form>

Notre formulaire comporte désormais des champs mais si vous essayez de lancer ce code vous remarquerez qu'il manque deux choses : un petit texte pour indiquer ce qu'il faut remplir dans le champs et un bouton pour envoyer toutes nos infos. Nous allons les ajouter maintenant.

Pour le texte nous allons utiliser la balise <label> pour ajouter du texte, il suffit juste de mettre cette balise devant notre input avec le texte que l'on souhaite 

                        
<label>Votre nom</label> <input type="text" name="Nom" />

Nous avons maintenant un petit texte devant notre champ qui indique ce qu'il faut remplir

Passons au bouton, la balise que nous allons utiliser est encore la balise input
mais avec le type "submit" 

                        
<input type="submit" name="bouton" value="envoyer" />

Value permet de définir le texte a l'intérieur du bouton

                        
<form method="post" action="formulaire.php">
    <label>Votre nom</label>   <input type="text" name="Nom" />
    <br>
    <label>Votre prenom</label><input type="text" name="prenom" />
    <br>
    <label>Votre email</label> <input type="email" name="mail" />
    <br>
    <input type="submit" name="bouton" value="envoyer" />
</form>

Voici a quoi doit ressembler votre code, enregistrons le et lancons notre site

Voilà a quoi ça devrais ressembler, maintenant nous allons faire en sorte que notre formulaire puisse envoyer ces infos et de pouvoir les afficher.

Création de la page PHP

Pour afficher ces infos, nous allons créer une nouveau fichier qui sera cette fois çi au format PHP et qui a le même nom que le fichier php que vous appelez dans votre formulaire. Nous allons construire ce fichier php comme une page html avec un head un body etc etc.
Nous allons ensuite dans le body mettre une balise <p> qui va afficher un simple texte et dans cette balise p nous allons ouvrir une nouvelle balise réservé au php : la balise <? php ?> 

                        
<?php ?>

Dans cette balise nous allons utiliser une fonction appelée echo qui permet d'afficher du texte. Nous allons ensuite demander a echo d'afficher une information que nous avons récupéré via la method post, ce qui se traduit par $_POST['Nom_Du_Champ']; .Dans le Nom_Du_Champ on mettra le nom du input qu'on a mis dans l'argument name. 

                        

<p> Votre nom est <?php echo $_POST['nom']; ?> </p>
<p> Votre prenom est <?php echo $_POST['prenom']; ?> </p>
<p> Votre email est <?php echo $_POST['email']; ?> </p>

Voilà a quoi doit ressembler le contenu de votre body dans votre fichier php
Vous pouvez enregistrer le code php et revenir sur votre formulaire, remplissez le et cliquez sur le bouton envoyer.

Et voilà le travail. C'est du PHP extrêmement simple mais vous avez réussi a transférer des informations d'une page a l'autre ce qui est déjà pas mal !
