Premiers préparatifs

Pour faire un site digne de ce nom, il faut en premier quelques outils :
-Un editeur de texte, je vous conseille VS Code
-Un navigateur Internet (mais vu que vous êtes sur ce site j'imagine que vous en avez un).
-Et un peu de patience

Création du fichier

Pour créer une page web, nous allons ouvrir VS Code et créer un nouveau fichier en cliquant sur Fichier puis sur Nouveau fichier (ou vous pouvez utiliser le raccourci Ctrl N)

Voici ou vous devriez être


Il vous suffit ensuite de cliquer sur Sélectionner un langage et de choisir "HTML"

Ok, tout ça c'est cool, mais c'est quand qu'on code ?

Et bien maintenant justement.
Nous allons poser les première briques de notre page web !

Les Balises

Pour créer une page web nous allons utiliser ce qu'on appelle des "Balises". Dans ces Balises Nous allons rentrer les informations de notre site. Je vais vous donner un exemple simple de balise avec la balise <p> qui représente en HTML un paragraphe.

Une balise est composé d'une balise ouvrante et fermante :
Une balise ouvrante s'écrit avec des chevrons et le type de balise ( <p> )
Et une balise feramnte avec elle aussi des chevrons mais avec un / pour indiquer
qu'elle se ferme (</p>)

Donc si on veut écrire "Les pâtes a la carbonara sont les meilleures", on va écrire :

                    
<p>Les Pâtes a la carbonara sont les meilleures</p>

Il existe aussi des balises dites "Orpheline" qui s'ont pas besoin de se fermer avec une balise fermante comme la balise <img> qui permet d'afficher une image. Il suffit juste de rentrer les informations dans la balise et le tour est joué.

Il suffit donc de mettre uniquement des paragraphes
et on peut créer un site comme ça ?

Pas exactement...

Pour créer un site Web il faut respecter une structure de page pour faire en sorte que le code soit propre, structuré et que tout fonctionne correctement. Et il existe aussi une multitude de balises que nous n'avons pas encore vu. Nous allons voir ça maintenant

Création de la page Web (enfin)

Il est maintenant temps de débuter la création de la page en elle même ! Pour le moment nous allons rester simple et créer une page web qui affichera la phrase de votre choix
(si possible sans grossièreté, mais c'est vous qui voyez.)


Présentation du Code

                
<!DOCTYPE html>
<html>
<head>
    <title> Hello World </title>
    <meta charset="utf-8">
</head>
<body>
    <p> Hello World </p>
</body>
</html>

Voici donc a quoi va ressembler votre code, je vais maintenant vous expliquer la fonction de chaque balise, vous pouvez taper le code en même temps que vous lisez la fonction des balises (ou faire juste un copier coller et lancer la page, vous aurez déjà adopté les bonnes, ou mauvaises, manières d'un développeur) 

                
<!DOCTYPE html>

La balise Doctype indique a notre navigateur Internet que la page que nous souhaitons afficher et de type HTML. Vous pouvez aussi remarquer que c'est une balise Orpheline 

                
<html> </html>

La balise html va contenir absoument tout notre code notre code, on va donc mettre toutes nos balises Dans la balise html.


Ok, maintenant que nous avons notre balise html, nous allons diviser notre code en deux grandes parties: Le Head et le Body.

Ah c'est donc ça la fameuse "Structuration"

Et oui, dans notre balise head nous allons rentrer toutes les informations qui sont importentes pour la page qui ne sont pas affichées a l'utilisateur tel que le titre de l'onglet, la taille de la police etc etc. Et pour le body c'est tout ce qui va être afficher a l'utilisateur, on va donc retrouver notre texte, nos images etc etc.


Commençons par la balise head. Nous allons utiliser deux balises, la balise title et la balise meta charset


                    
<title> </title>

La Balise Title va indiquer le nom de l'onglet de la page web sur notre navigateur.

                    
<meta charset="utf-8">

La Balise meta charset permet d'indiquer au navigateur quel type de caractère il doit afficher sur notre Page web. La plus part du temps vous allez utiliser l'argument utf-8. 

                    
<head>
    <title> Hello World </title>
    <meta charset="utf-8">
</head>

Voici a quoi ressemble donc notre head pour le moment. Dans mon example quand nous allons ouvrir notre page web, le titre de l'onglet sera "Hello World"


Nous allons maintenant passer au Body
Et ça va être très court car nous allons utiliser pour le moment qu'une seule balise, la balise p 

                    
<p> </p>

La balise p permet d'afficher du texte dans notre page, il suffit juste de mettre ce que l'on veut Dans la balise comme dans l'exemple des pâtes dans la section au dessus. 

                    
<body>
    <p> Hello World </p>
</body>

Et voici le body. N'oubliez pas que le head et le body doivent être tout les deux dans la balise html.


Voici donc a quoi votre code doit ressembler pour le moment. Il est venu le moment de le tester !

Affichage de la Page Web

Maintenant que votre code est prêt, il est l'heure de l'exécuter ! Vous allez devoir en premier l'enregistrer en cliquant sur fichier puis sur enregistrer ou en utilisant le reccourci Ctr s.

Pour le nom, vous pouvez l'appeler comme vous voulez, mais assurez vous juste que le fichier soit bien au format .html

Vous devriez normalement vous retrouvez avec ce fichier, le logo dépendra de votre navigateur internet pas défault (pour ma part chrome). Si jamais vous ne pouvez pas l'ouvrir, il vous suffit de faire clique droit, ouvrir avec et de sélectionner votre navigateur internet.



Et voila le travail !
