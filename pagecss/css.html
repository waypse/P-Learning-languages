CS quoi ?

Vous savez maintenant mettre du contenu dans une page, c'est déjà pas mal ! Mais désormais il va falloir l'egayer un peu sinon votre site va être bien triste ! C'est le travail du CSS de donner du style a votre Site, nous allons faire quelque chose de très simple pour le moment.

Connexion d'une page css a notre page Web

Pour cet exercice nous allons reprendre notre site web de la section précédente et nous allons rajouter une nouvelle balise dans la section head. Cette balise c'est <link> (non, pas le personnage) 

                        
<link rel="stylesheet" href="Nom_Du_Fichier_CSS.css">

Cette balise permet d'indiquer a notre page html qu'il doit utiliser un fichier CSS pour gérer les éléments graphiques de la page telle que la couleur ou bien la position. rel permet de définir le type de fichier que nous utilisons, dans notre exemple "stylesheet" ou dans la langue de Cyril Hanouna "Feuille de Style". Quand a href, il indique le chemin pour accéder a notre fichier css. Si on ne met pas de chemin et que l'on met directement le nom du fichier, il faut que le fichier html et css soient dans le même dossier.


Nous allons donc mettre cette balise dans notre head

                        
<head>
<title> Hello World </title>
<meta charset="utf-8">
<link rel="stylesheet" href="style.css">
</head>

Voilà, nous avons indiqué a notre page web qu'il doit utiliser le fichier "style.css" comme feuille de style pour les éléments de la page.
Passons a la pratique !

Utilisons toutes les couleurs !

Maintenant que notre page est prête a utiliser notre fichier css, nous allons donc devoir le créer, pour faire ça, nous allons faire exactement comme nous avons fait pour créer le fichier html :
-Dans Vscode Cliquer sur Fichier / Nouveau Fichier
-Quand le fichier est créer cliquer sur Sélectionnez un langage et choisir CSS

La façon d'écrire du CSS n'est pas la même que sur HTML, on retrouve un système similaire au balise mais pas écrit de la même façon, nous allons utiliser les { } (oui le truc que tu arrivais jamais a écrire a l'école) 

                        
Element_a_modifier {
modification : valeur de la modification;
}

Et pour le nom des balises, ce sera l'élément que nous souhaitons modifier. Petit exemple, nous voulons modifier la couleur de notre phrase dans notre page web pour qu'elle soit rouge, nous allons donc écrire 

                        
p {
color: red;
}

Essayez donc ! Utilisez ce code CSS, sauvegardez votre fichier css avec le nom style.css (faites en sortes quel le fichier html et css soient dans le même dossier C'EST TRES IMPORTENT)

Voilà a quoi ça devrais ressembler !

Les Classes

Mais du coup, si je modifie l'élément p, cela va modifier tout le texte de ma page web ?

Exactement ! C'est pour ça que le système de classe existe pour pouvoir modifier un seul élément en particulier. Nous allons reprendre notre page web et y rajouter une nouvelle balise <p> et nous allons lui ajouter une classe pour que le texte soit de couleur bleu a la place de rouge. 

                        
<p class="phraseBleu"> Hello World mais en bleu ! </p>

Comme vous pouvez le voir (sauf si vous êtes aveugle) nous avons rajouter a l'intérieur même de notre balise ouvrante l'argument "class" et nous l'avons appeler "phraseBleu", c'est cette classe que Nous allons utiliser dans notre css pour pouvoir modifier en particulier cet élément.

Retournons dans notre CSS et appelons notre classe.

                        
p {
color: red;
}

.phraseBleu {
color: blue;
}

Pour pouvoir appeler une classe personalisée, il faut mettre un point devant le nom de la classe que l'on souhaite appeler (donc dans notre exemple .phraseBleu).

Enregistrons donc notre code CSS et testons donc.

Bingo

Voila, vous savez désormais modifier la couleur d'un élément, mais tout comme le HTML, le CSS est un outil extrêmement puissant qui permet de faire passer une simple liste de course en un véritable site internet du 21ème siècle.
