Créer un site web en
HTML CSS JavaScript et PHP

Un site web ou simplement site, est un ensemble de pages web et de ressources reliées par des hyperliens, défini et accessible par une adresse web. Un site est développé à l'aide de langages de programmation web, puis hébergé sur un serveur web accessible via le réseau mondial Internet, un intranet local, ou n'importe quel autre réseau.

HTML filter_1

HTML est un markup language (langage de marquage ?). Il permet d'enrichir un texte avec des informations structurelles, sémantiques et de présentation. Le principe de HTML, commun à ce type de langages, consiste à utiliser des éléments délimités par des balises.

CSS filter_2

Le CSS permet de mettre en page le contenu (emplacement des éléments), et mettre en forme le texte (police d'écriture, couleur, taille).

PHP filter_4

Le PHP est un langage dis "Back-End" qui va permettre le transfert d'information d'une page à une autre dans notre site.

JavaScript filter_3

Le JavaScript permet d'ajouter du dynamisme, mais uniquement du côté du navigateur (par exemple des animations graphiques). Il ajoute du dynamisme à notre site.
